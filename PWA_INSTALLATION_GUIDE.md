## What is a PWA?

A **Progressive Web App (PWA)** allows you to install the admin dashboard as a standalone app on your device. Benefits include:

‚úÖ **Quick Access**: App icon on your home screen  
‚úÖ **Offline Support**: View cached data without internet  
‚úÖ **Full Screen**: No browser UI, feels like a native app  
‚úÖ **Push Notifications**: (Can be enabled in future)  
‚úÖ **Fast Loading**: Cached assets load instantly

## üì± iOS Installation (iPhone/iPad)

### Requirements
- iOS 11.3 or later
- Safari browser (required)

### Step-by-Step Instructions

1. **Open Safari** on your iPhone/iPad

2. **Navigate to your admin dashboard**:
   ```
   https://your-wedding-site.vercel.app/admin
   ```

3. **Tap the Share button** 
   - Look for the square icon with an arrow pointing up
   - Located at the bottom of the screen (iPhone) or top (iPad)

4. **Scroll down** in the share menu

5. **Tap "Add to Home Screen"**
   - You'll see the app icon preview
   - Default name: "Wedding Admin"

6. **Customize (optional)**:
   - Change the app name if desired
   - The icon is pre-configured with your wedding colors

7. **Tap "Add"** in the top right

8. **Done!** 
   - The app icon appears on your home screen
   - Tap it to open the admin dashboard as a standalone app

## ü§ñ Android Installation

### Requirements
- Android 5.0 (Lollipop) or later
- Chrome browser (recommended)

### Step-by-Step Instructions

1. **Open Chrome** on your Android device

2. **Navigate to your admin dashboard**:
   ```
   https://your-wedding-site.vercel.app/admin
   ```

3. **Tap the menu** (three vertical dots in top right)

4. **Tap "Add to Home screen"** or **"Install app"**
   - Chrome will show an install prompt
   - You may see "Install" banner at the bottom

5. **Confirm installation**:
   - Tap "Install" or "Add"
   - The app will be added to your home screen

6. **Done!**
   - Find the app icon in your app drawer
   - Tap to open as a standalone app

### Alternative Method (Chrome Banner)

Chrome may automatically show an install banner:
- Look for "Add Wedding Admin to Home screen" at the bottom
- Tap "Add" to install instantly

## ‚ùì Troubleshooting

### "Add to Home Screen" not showing (iOS)

**Solutions**:
- ‚úÖ Use Safari (not Chrome or other browsers)
- ‚úÖ Make sure you're on the `/admin` page
- ‚úÖ Check iOS version (need 11.3+)
- ‚úÖ Try refreshing the page

### Install prompt not appearing (Android)

**Solutions**:
- ‚úÖ Use Chrome browser
- ‚úÖ Visit the site a few times first
- ‚úÖ Use the manual method (Menu ‚Üí Add to Home screen)
- ‚úÖ Check Chrome is up to date

### App not working offline

**Expected behavior**:
- First visit requires internet to cache data
- Subsequent visits work offline
- New data requires internet connection

### App icon looks wrong

**Solution**:
- Uninstall and reinstall the app
- Clear browser cache first
- Icons are auto-generated at 192x192 and 512x512
